# 题材结构模式

## 目录

- 小说与叙事作品
- 思想型非虚构作品
- 历史与传记
- 混合型作品
- 结构选择检查

## 小说与叙事作品

### 主线

用最少的人物和支线保留完整的因果链，让听众既能跟上故事，又能理解作品为什么这样写。默认允许完整剧透，因为任务是交付核心内容；如果用户明确要求不剧透，再改变揭示范围。

### 推荐结构

1. 开场：交代作品类型、作者和最有吸引力的冲突。只给理解故事必需的背景。
2. 第一部分：建立世界、人物欲望和触发事件，让核心问题真正发生。
3. 第二部分：沿关键选择和后果推进冲突。压缩重复事件和次要人物，不跳过决定结局的因果环。
4. 第三部分或第四部分：完成高潮与结局，再解释主题、叙事设计、历史真实性或作品影响。
5. 结尾：从结局回看人物和主题，不把故事变成空泛的人生鸡汤。

故事通常占正文约六成到八成，分析占两成到四成。悬疑、冒险和情节驱动作品可以提高故事比例；文学性强、结构复杂的作品应增加叙事手法和主题分析。

### 压缩规则

- 只保留会改变主角选择、关系或结局的人物。首次出现时用一个辨识度高的特征帮助记忆。
- 用场景承载关键转折，不把整本小说写成事件清单。
- 每次时间跳跃、视角变化或世界切换都给听众一个清楚的定位句。
- 讲清剧中人物当时知道什么，不用结局信息污染前面的悬念。
- 在关键阶段回放中同时核对人物能力与状态变化，例如是否识字、受伤、掌握某项信息、仍与谁保持联系；不能为了叙事顺滑把早期状态沿用到成年或后期。
- 对每个决定高潮或结局的动机与因果标记证据层级：文本明示、叙述者或角色推测、讲稿解释。后两类必须保留归属和不确定措辞；多个角色给出竞争性解释时，不能替原著裁决。
- 人物做了某类工作、参与某次设局或拥有某种权限，不自动证明其正式身份、组织成员资格或完整动机；原著只拒答一个问题时，不得把拒答范围扩展到相邻人物或组织。主线依赖这些连接时，把连接本身列为桥接主张核验。
- 把创作时间、出版时间和故事发生时间分别记录。正文出现年份时必须说清它属于哪一种时间。
- 区分“小说中设定如此”和“现实中确实如此”。涉及科学、历史或真实人物时单独核验。
- 完稿后按“相识、升级、预警、发现、高潮、结局”回放关键阶段，检查压缩有没有把相隔较远的事件粘成同一阶段。

## 思想型非虚构作品

适用于商业、经济、科学、哲学、心理学、社会科学和方法类著作。

### 主线

把书重组为一个中心问题和两到四个相互连接的答案。不能只列观点；必须重建作者怎样从问题走到结论。

### 推荐结构

1. 开场：用现实困惑、反常识现象、实验或作者遭遇引出中心问题。
2. 第一部分：说明旧解释为什么不够，建立理解后文所需的核心概念或机制。
3. 第二部分：用书中最有解释力的案例、数据或思想实验展开核心论证。
4. 第三部分：处理应用、推论、反例、限制或争议，让观点不悬空。
5. 结尾：重答中心问题，说明哪些认识可以带走，以及它们在什么条件下成立。

### 解释规则

- 每个核心观点按“问题、作者答案、依据、白话解释、边界”展开。
- 动笔前建立“具体证据载荷”：经验研究型作品至少保留一项研究方法与结果、两个能推进论证的原书案例和一个研究边界；哲学或纯理论作品用一个具名思想实验、文本论证或对话场景替代实证研究。三个生产块各预放至少一个载荷，不能写到最后才发现全稿只有概念。
- 具体材料及其解释原则上占主体约三成五到五成。连续两段抽象解释后必须进入已核验的研究现场、人物、案例、思想实验或反例；“一个人”“某位研究者”“某家公司”这类无来源泛化角色不能冒充原书案例。
- 对因果主张说明中间机制，不把相关性直接说成因果。
- 科学假说要标明它是主流结论、有力解释、竞争性理论还是作者推测。
- 商业和投资观点要说明适用条件、失败风险和幸存者偏差，避免收益承诺。
- 心理学实验、统计数字和名人轶事是高风险材料；找不到可靠来源就不用精确值或直接删掉。
- 对研究、金融、政策与科学结论逐句保留极性和范围。重点反查未与已、最低与全额、加入与退出、增加与降低、相关与因果；医学和基因技术必须说清体细胞、胚胎、生殖系或可遗传层级，不能用未限定的“共识”跨层概括。
- 从物理过程推到记忆、因果或主观经验，从生物机制推到社会行为，从观察相关推到政策效果，或从事实描述推到价值判断时，把中间连接当作独立的高风险桥接主张。分别核验桥两端和桥本身；作者路线、有力解释、竞争方案与当前共识必须分开。
- 书内术语、作者在其他作品中的术语和写作者自拟概括分别标记；只验证到作者相关思想时，不能声称该术语就在本书中。
- 应用不能脱离原书强行上价值。只有当推论确实由前文建立时，才延伸到个人生活。
- 写完提纲后对照可靠目录做覆盖审计。主线之外若仍有一个相对独立且影响全书含义的论证单元，应纳入正文，或明确把它与某一主观点合并，而不是悄悄遗漏。
- 书外应用要用自然措辞标明是沿着机制所作的延伸，不要使用“作者告诉我们”把自己的推论归给作者。

## 历史与传记

### 主线

不要把年代和事件依次念一遍。选择一个结果、困境或人物变化作为问题，通过关键决策、制度条件和偶然因素建立因果链。

### 推荐结构

1. 开场：从反常结果或关键时刻切入，例如“看起来占优势的一方为什么失败”。
2. 第一部分：交代时代结构、人物起点和冲突条件。
3. 第二部分：选择两到四个决定走向的节点，讲清当事人的信息、选择、代价和后果。
4. 第三部分：总结深层机制，并比较书中解释与其他合理解释。
5. 结尾：说明这段历史或这个人物为什么仍值得理解，避免简单贴英雄或反派标签。

### 取舍规则

- 年份只在定位、因果或先后关系需要时出现。
- 主题重组可以改变叙述顺序，不能暗改事实顺序。直接连接两个历史事件节点时，“接下来、随后、后来、此后、再往后”等属于时间前进断言；节点时间回退时，必须在新节点首句用“此前、在这之前、这里回到……”明确标记回叙。同期使用“与此同时、同一阶段”等定位；顺序未定时使用中性主题转场，不得暗示先后。
- 头衔和亲属关系第一次出现时说清，后面使用稳定称呼。
- 区分作者的判断、史料记载和后世传说。
- 传记既讲成就，也讲方法形成的过程、限制和失败。不要把财富、地位或名气自动解释为品格证明。
- 对有争议的人物和时期，至少核对一种不同立场，正文可以简要呈现争议而不必追求虚假平衡。
- 对照目录逐章建立覆盖地图，每个独立章节都必须记录“进入主线、与哪一章合并、或为何不影响中心答案”。至少选择三个来自不同章节的已核验节点，按“当事人当时知道什么、作了什么选择、付出什么代价、造成什么后果”写成具体材料；三个生产块各锚定一个节点。
- “所知、选择、代价、后果”只用于事实底稿核对。正文要把它们讲成自然场景和因果，不得反复念出这组规划标签或套用同一句模。
- 先找出人物或制度走向变化的直接导火索，再解释深层结构。不能只写宏观机制而漏掉作者用来启动因果链的继承争议、任免、战役、审判、疾病或关系破裂等具体事件。
- 具体人物、事件与选择原则上占主体约一半。不能用“信息、流程、资源、系统、组织能力”等泛化管理词替代原书的关键史实，也不能把人物的矛盾立场压成单一标签。

## 混合型作品

纪实文学、回忆录、寓言式商业书和以故事解释科学的作品往往同时属于两个轨道。选择一个主轨道决定全稿节奏，再用另一轨道补足：

- 故事是论证工具时，以思想型非虚构为主，不必复述全部情节。
- 人物经历本身就是价值所在时，以叙事或传记为主，在关键节点解释观点。
- 历史材料服务于一个理论时，以中心问题为主，只保留支撑机制的年代和事件。

## 结构选择检查

动笔前回答：

1. 听众听完后最应该能回答哪一个问题？
2. 哪三个左右的内容单元足以回答它？
3. 每个单元最有力的具体材料是什么？
4. 哪些著名内容虽然有趣，却不服务这条主线？删掉它们。
5. 结尾要回收哪一个开场承诺？
