## 二、模块角色速览

映射依据：以各模块的数据处理角色（是否面向用户、采集何种数据）决定基线适用性。
┌─────────┬────────────────────┬──────────────┬──────────────────────────────────────┐
│  模块   │    数据处理角色     │ 是否面向用户 │           采集数据类型               │
├─────────┼────────────────────┼──────────────┼──────────────────────────────────────┤
│ CDC     │ 座舱交互中枢        │ ✅ 有屏+语音 │ 账号/位置/联系人/语音/偏好/舱内图像   │
│ ADC     │ 智驾感知与决策      │ ❌ 间接(HMI) │ 车外视频/雷达点云/定位/DMS           │
│ Tbox    │ 通信网关            │ ❌ 无UI      │ 车辆状态/位置/CAN/OTA                │
│ TSP     │ 云端平台            │ ❌ 后端       │ 所有数据汇聚/分析/AI训练              │
│ 手机app │ 车主移动端          │ ✅ 高频交互  │ 手机号/位置/车辆控制/行程/支付        │
│ 生态app │ 第三方车载应用      │ ✅ 间接      │ 内容偏好/播放记录/账号关联            │

│ ZCM     │ 区域车身控制        │ ❌ 无UI      │ 车速/制动/转向/门锁/灯光运行数据      │
│ 小程序  │ 车载轻应用          │ ✅ 间接      │ 账号/使用记录/位置(授权后)            │
│ 销服    │ 销售与售后系统      │ ✅ 独立交互  │ 身份证/联系方式/VIN/维保/金融         │

是，不存在

不涉及
